Les préférences du module FAQ

Lorsque le module FAQ est déployé au niveau de la colonne gauche de l’applicatif, l’icône s’affiche.

Elle permet de définir les préférences du module.

  • Remplir les champs relatifs aux préférences concernant l’affichage des questions-réponses :
    • « Nombre de questions-réponses par page » : Pour définir le nombre de questions-réponses maximum par page à afficher.
    • « Texte pour dossier vide » : Champ textuel libre pour indiquer le texte par défaut s’affichant si aucune question-réponse n’est disponible sur la page listant les questions-réponses.
    • « Formulaire affiché » : Liste déroulante « oui / non » permettant de faire s’afficher ou non le formulaire permettant à l’internaute de proposer une question via le site.

Une fois l’ensemble des champs remplis :

  • Cliquer sur pour enregistrer les préférences du module « FAQ ».
  • Cliquer sur pour enregistrer les préférences du module FAQ tout en fermant ensuite la fenêtre des préférences.
